# Test Repo

Prototype assets for a TMJ digital therapeutics platform consisting of:

- **AI Strategy** (`docs/tmj_ai_strategy.md`): outlines model choices, fine-tuning,
  and deployment for pose estimation, action-quality scoring, and coaching.
- **Clinician Portal** (`doctor-portal/`): FastAPI backend + React front-end for
  managing treatment exercises, prescribing videos, and reviewing adherence.
- **Guidance Workflow** (`docs/threshold_guidance.md`): details how clinician
  thresholds translate into AI-driven coaching.
- **Patient Mobile App** (`patient-app/`): Flutter prototype that lists assigned
  exercises, streams camera footage, and shows live feedback stubs.
